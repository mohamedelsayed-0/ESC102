How These Sources Are Used

Traceability notes

Project pages

Interpretation first, source attached

The project sections do not reproduce entire reports. They interpret design decisions, then point back to the artifact that supports that interpretation.

CTMF handbook

Concepts stay tied to evidence

The CTMF entries are only included when I can point to a project move, artifact, or credible retrospective lesson that justifies the concept being there.

Generated files

Some references are source files, others are outputs

The Praxis II dashboard, threshold charts, and sensitivity atlas are generated outputs. They are listed beside the scripts and data files that produced them so the final visual does not have to stand on trust alone.

Portfolio figures

Site images are treated as evidence pointers, not standalone sources

The PNG and SVG figures used across the pages are included here because they help readers find the relevant part of a larger report or codebase quickly, but they always sit beneath a fuller source file.
