Projects featured here

Praxis I

Backpack Strap Design

Team-based redesign work focused on excess strap snagging, adjustability, comfort, and manufacturability.

CTMFs carried by this case

  • Design concepts Three directions stayed distinct long enough to expose different tradeoffs.
  • Evaluation criteria Snagging, adjustability, comfort, and manufacturability became explicit criteria.
  • Proxy testing A simplified simulation acted as a low-resource stand-in for full ergonomic testing.

CIV102

Bridge Design

Matboard bridge design supported by drawings, calculations, and analysis code for load envelopes and failure modes.

CTMFs carried by this case

  • Requirements Span, load, material rules, and fabrication limits stayed active throughout.
  • Verification Calculations, envelopes, and drawings checked whether the bridge survived build reality.
  • Prototypes: graphical The drawing package worked as a graphical prototype for spacing and assembly.

Praxis II

Daycare Wristband Workflow

Team-based workflow redesign that converged on a parent-led two-station system with a loading-aid prototype.

CTMFs carried by this case

  • Evaluation criteria The requirement table converted workload, safety, intuitiveness, and setup into metrics.
  • Workflow prototype The direction became clearer once the device and parent flow were read together.
  • Sensitivity analysis Validation results showed where the chosen direction stayed useful.

Independent Study

Gaussian Entanglement Project

Analytic and computational work on entanglement thresholds in symmetric phase-insensitive Gaussian channels.

CTMFs carried by this case

  • Analytical The central move was turning the entanglement condition into an analytical claim.
  • Toulmin’s structure of argument The derivation, figures, and code kept the claim, grounds, and warrant visible.[24]
  • PIAA The plots mattered once they were used to revise the claim.
