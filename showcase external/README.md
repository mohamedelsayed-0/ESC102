Showcase visuals bundle for the Wrist Wizard simulation work.

Contents:
- `showcase.html`: local viewer page for the final animation
- `wrist-wizard-showcase.mp4`: final showcase video
- `wrist-wizard-showcase-poster.png`: poster frame for the video
- simulation comparison and heatmap SVGs used for showcase visuals

Open `showcase.html` in a browser or serve this folder locally to view the animation with its linked assets.
