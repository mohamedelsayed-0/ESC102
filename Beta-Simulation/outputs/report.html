Daycare Wristband Prototype Report

This report compares the current wristband process against two proposed alternatives: parents applying wristbands before class check-in, and a dispensing machine with one or two stations.

Current process employee time 34m 30s

Assumptions: 90 children, 10 classrooms, a 30-minute arrival window peaking 9 minutes before departure, and 400 Monte Carlo runs per point.

Decision Summary

Parent model

Useful from about 28% compliance

Staff time uses 5 seconds per child if compliant, plus 25 extra seconds for each missed or incorrect wristband.

1 machine

Useful until about 70.5% struggle

This is the struggle-rate limit before machine cases consume more employee time than the current process.

2 machines

Useful until about 70.5% struggle

Under employee time, the two-machine line overlaps the one-machine line because staff work happens after the machine handoff fails.
