Daycare Wristband Simulation

Monte Carlo comparison of the current wristband process against two proposed prototypes: parent-applied wristbands and a dispensing machine with one or two stations.

Decision metrics

Parent: employee time. Machine: readiness time.

The parent model is useful when it saves employee time. The machine model is useful when it gets everyone ready faster than the current process.

Assumptions

Parent compliance now measures employee time directly: 5 seconds per child for attendance/check, plus 25 extra seconds for each failed parent case. Machine charts still measure total readiness time because the queue happens at drop-off.
